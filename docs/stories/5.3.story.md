# Story 5.3 — Integração NuvemShop Blog

**Epic:** 5 — Content Engine: Automação e Integração
**Status:** Ready
**Prioridade:** P2
**Executor:** @dev
**Estimativa:** M
**Depende de:** 5.2

---

## Descrição

Como Mauro, quero que após minha aprovação no Cockpit o post seja publicado automaticamente no blog da NuvemShop da Plaka, com imagem e alt text corretos, sem que eu precise abrir o painel da NuvemShop.

---

## Acceptance Criteria

- [ ] AC1: Após aprovação no Cockpit, o post é publicado no blog da NuvemShop via API
- [ ] AC2: Post publicado inclui: título, corpo em HTML, meta description, imagem com alt text
- [ ] AC3: A imagem é buscada do catálogo NuvemShop da Plaka (produto relevante ao tópico) ou placeholder se não encontrada
- [ ] AC4: Status no Cockpit atualiza para `publicado` com link direto ao post
- [ ] AC5: Em caso de falha na API NuvemShop, status vai para `erro_publicacao` com mensagem descritiva
- [ ] AC6: Credenciais NuvemShop armazenadas como variáveis de ambiente (`NUVEMSHOP_ACCESS_TOKEN`, `NUVEMSHOP_USER_ID`)

---

## Escopo

**IN:**
- Publicação via NuvemShop API (endpoint de criação de posts do blog)
- Busca de imagem do catálogo por relevância ao tópico
- Conversão do Markdown do Lyra para HTML
- Log de publicação com URL do post

**OUT:**
- Agendamento de publicação (publica imediatamente após aprovação)
- Categorias e tags avançadas no blog
- SEO schema markup automático (fase futura)

---

## Notas Técnicas

- NuvemShop API: `POST /blogs/{blog_id}/articles`
- Converter Markdown → HTML antes de enviar
- Imagem: buscar produto da coleção mencionada no briefing ou mais recente
- Referência: NuvemShop Partner API docs
