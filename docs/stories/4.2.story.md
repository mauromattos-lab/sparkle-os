## Riscos

- **git log em Windows:** `execSync('git log ...')` funciona em bash (WSL) mas pode ter problemas de encoding no Node.js Windows. Usar `{ encoding: 'utf8' }` no execSync.
- **Stories sem frontmatter:** Arquivos em `docs/stories/` sem frontmatter YAML válido devem ser ignorados silenciosamente (try/catch por arquivo).
- **Volume de commits:** Repositório grande pode ter muitos commits. Limitar a 20 commits e last 48h para stories.

Extração de frontmatter simples (sem biblioteca externa):
```typescript
function parseFrontmatter(content: string): Record<string, string> {
  const match = content.match(/^---\n([\s\S]*?)\n---/);
  if (!match) return {};
  const result: Record<string, string> = {};
  for (const line of match[1].split('\n')) {
    const [key, ...rest] = line.split(':');
    if (key && rest.length) result[key.trim()] = rest.join(':').trim().replace(/^["']|["']$/g, '');
  }
  return result;
}
```
