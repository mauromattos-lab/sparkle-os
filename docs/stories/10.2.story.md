# Story 10.2 — API: endpoints /conversations e /metrics por cliente

## User Story
Como cockpit do cliente, quero endpoints autenticados que retornem as conversas e métricas do meu tenant, para que o frontend possa exibir dados reais sem expor informações de outros clientes.

## Acceptance Criteria
- [ ] 1. `GET /api/client/conversations` retorna lista paginada de conversas do tenant autenticado
- [ ] 2. `GET /api/client/metrics` retorna totais: conversas hoje, conversas totais, status do sistema
- [ ] 3. Middleware de auth valida token Supabase Auth e injeta `isolationKey` na request
- [ ] 4. Sem token válido → 401. Token válido mas sem cliente vinculado → 403
- [ ] 5. Dados de um tenant nunca aparecem na resposta de outro tenant (RLS garantido)
- [ ] 6. Regra Sparkle-first: endpoints testados com dados reais da instância SparkleOS

## Dev Notes

**Middleware de auth:**
```typescript
// apps/core/src/middleware/client-auth.ts
import { createClient } from '@supabase/supabase-js';
import { getClientSession } from '../../../organs/zenya/src/auth/session';

export const clientAuthMiddleware = async (c: Context, next: Next) => {
  const token = c.req.header('Authorization')?.replace('Bearer ', '');
  if (!token) return c.json({ error: 'UNAUTHORIZED' }, 401);

  const { data: { user }, error } = await supabase.auth.getUser(token);
  if (error || !user) return c.json({ error: 'UNAUTHORIZED' }, 401);

  try {
    const session = await getClientSession(user.id);
    c.set('clientId', session.clientId);
    c.set('isolationKey', session.isolationKey);
  } catch {
    return c.json({ error: 'CLIENT_NOT_FOUND' }, 403);
  }
  await next();
};
```

**Endpoint de conversas:**
```typescript
// GET /api/client/conversations?limit=20&offset=0
app.get('/api/client/conversations', clientAuthMiddleware, async (c) => {
  const isolationKey = c.get('isolationKey');
  const limit = Number(c.req.query('limit') ?? 20);
  const offset = Number(c.req.query('offset') ?? 0);

  // set_config obrigatório antes de query em zenya_conversations
  await db.execute(sql`SELECT set_config('app.current_client_key', ${isolationKey}, TRUE)`);
  const rows = await db.select().from(zenyaConversations)
    .orderBy(desc(zenyaConversations.createdAt))
    .limit(limit).offset(offset);

  return c.json({ data: rows, limit, offset });
});
```

**Endpoint de métricas:**
```typescript
// GET /api/client/metrics
app.get('/api/client/metrics', clientAuthMiddleware, async (c) => {
  const isolationKey = c.get('isolationKey');
  await db.execute(sql`SELECT set_config('app.current_client_key', ${isolationKey}, TRUE)`);

  const [total, today] = await Promise.all([
    db.select({ count: count() }).from(zenyaConversations),
    db.select({ count: count() }).from(zenyaConversations)
      .where(gte(zenyaConversations.createdAt, startOfDay(new Date())))
  ]);

  return c.json({
    totalConversations: total[0].count,
    conversationsToday: today[0].count,
    systemStatus: 'active'
  });
});
```
