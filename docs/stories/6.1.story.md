## Notas Técnicas

### Instalação Ghost (opção recomendada — Ghost CLI)
```bash
npm install -g ghost-cli
mkdir -p /var/www/ghost && cd /var/www/ghost
ghost install --db sqlite3 --no-prompt --no-stack
ghost start
```

### Alternativa — Docker
```bash
docker run -d -e url=http://localhost:2368 \
  -p 2368:2368 \
  -v ghost-content:/var/lib/ghost/content \
  ghost:latest
```

### Ghost Admin API — autenticação com Staff API Key
```typescript
// Staff API Key format: {id}:{secret}
// JWT gerado localmente — sem chamada OAuth
import jwt from 'jsonwebtoken';

const [id, secret] = GHOST_ADMIN_API_KEY.split(':');
const token = jwt.sign({}, Buffer.from(secret, 'hex'), {
  keyid: id,
  algorithm: 'HS256',
  expiresIn: '5m',
  audience: '/admin/'
});

// POST /ghost/api/admin/posts/
```

### JSON-LD Schema BlogPosting (para AC4)
```json
{
  "@context": "https://schema.org",
  "@type": "BlogPosting",
  "headline": "{{título}}",
  "author": { "@type": "Organization", "name": "Plaka Acessórios" },
  "publisher": { "@type": "Organization", "name": "Plaka Acessórios" },
  "datePublished": "{{data}}"
}
```

### Variáveis de Ambiente necessárias
```
GHOST_API_URL=http://localhost:2368
GHOST_ADMIN_API_KEY={id}:{secret}  # gerado no painel Ghost > Integrations
```
