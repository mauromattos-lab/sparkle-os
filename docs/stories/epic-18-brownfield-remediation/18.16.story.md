## Acceptance Criteria

1. Migration 013 cria `zenya_execution_log` (granular por turn) + `zenya_ai_usage_daily` (rollup) — schema completo em DB Specialist Review §9

3. Job de rollup noturno calcula `ai_usage_daily` a partir de `execution_log`
4. 7 tools admin novas (UX Specialist Review §2.4) implementadas em `admin-agent.ts`:
   - `consultar_custo_mensal`

   - `listar_escalacoes_motivos`
   - `listar_falhas_recentes`
   - `listar_top_clientes`
   - `consultar_taxa_resolucao`
   - (manter `consultar_metricas`, `listar_conversas_abertas`, `listar_escaladas` existentes)
5. Densidade ≤2 msgs admin agent (regra explícita no prompt admin)
6. Smoke admin: Mauro pergunta "quanto custei essa semana?" → bot responde número + detalhamento opcional
