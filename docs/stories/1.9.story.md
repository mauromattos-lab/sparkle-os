# Story 1.9 — Mapa de Capacidades dos Agentes

## User Story
Como agente AIOS, quero um documento vivo com o mapa completo do time — quem existe, o que cada um faz, quais ferramentas e MCPs cada um pode usar — para que qualquer agente novo entre no sistema com contexto completo.

## Status
- [x] Draft
- [ ] Ready (pendente validação @po)
- [ ] InProgress
- [ ] InReview
- [ ] Done

## Acceptance Criteria
- [ ] 1. Todos os agentes AIOS documentados com escopo, responsabilidades e ferramentas disponíveis
- [ ] 2. MCPs e ferramentas mapeados por agente — inclui restrições (o que cada agente NÃO pode fazer)
- [ ] 3. Distinção clara e documentada: **agentes construtores AIOS** (máquina de Mauro, sessão Claude Code) vs **agentes workers dos Órgãos** (a ser construído nos Épicos futuros)
- [ ] 4. Processo de atualização documentado: quando novo agente ou ferramenta é adicionado, como o mapa é atualizado
- [ ] 5. Consultável por qualquer agente em qualquer sessão — arquivo em localização padrão e bem indexado
