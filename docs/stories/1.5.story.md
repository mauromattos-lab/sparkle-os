## Technical Context

### MCPs disponíveis no stack AIOS (Docker MCP Toolkit)
| MCP | Propósito | Como acessar |
|-----|-----------|--------------|
| **EXA** | Web search, pesquisa de concorrentes, notícias | `mcp__docker-gateway__web_search_exa` |
| **Context7** | Docs de bibliotecas (React, Supabase, Hono, etc.) | `mcp__docker-gateway__resolve-library-id` + `get-library-docs` |
| **Apify** | Web scraping, social media data | `mcp__docker-gateway__call-actor` |
| **Playwright** | Browser automation, screenshots, E2E tests | via playwright MCP |

### Credenciais necessárias para SparkleOS
| Serviço | Variável | Responsável |
|---------|----------|-------------|
| Supabase | `SUPABASE_URL`, `SUPABASE_ANON_KEY`, `SUPABASE_SERVICE_KEY` | Mauro fornece |
| Redis (Coolify) | `REDIS_URL` | @devops configura no Coolify |
| Claude / Anthropic | `ANTHROPIC_API_KEY` | Mauro fornece |
| VPS SSH | chave privada + `VPS_HOST`, `VPS_USER` | @devops configura em CI/CD |
| GitHub | `GITHUB_TOKEN` | @devops configura |

### Estrutura de `.env` para o projeto
```bash
# packages/core/.env.example (template sem valores reais)
SUPABASE_URL=https://xxx.supabase.co
SUPABASE_SERVICE_KEY=
REDIS_URL=redis://localhost:6379
ANTHROPIC_API_KEY=
PORT=3001
NODE_ENV=development

# apps/piloting/.env.example
NEXT_PUBLIC_API_URL=http://localhost:3001
SUPABASE_URL=
NEXT_PUBLIC_SUPABASE_ANON_KEY=
```
