# Story 7.7 — Ferramentas de Integração Customizada

**Epic:** 7 — Zenya como Órgão Nativo do SparkleOS
**Status:** Ready for Review
**Prioridade:** P2
**Executor:** @dev
**Estimativa:** M
**Depende de:** 7.3

---

## Descrição

**As a** sistema multi-cliente,
**I want** que novas integrações sejam criadas uma vez e ativadas por configuração,
**so that** qualquer cliente possa ter ferramentas específicas (Asaas, Loja Integrada, etc.) sem duplicação de código.

## Dev Notes

### O pattern — como criar uma nova integração

Este é o coração desta story. O padrão deve ser:

**1. Criar o arquivo de integração**
```typescript
// integrations/asaas.ts
import { tool } from 'ai';
import { z } from 'zod';

export function createAsaasTools(tenantId: string, asaasConfig: AsaasConfig) {
  return {
    criarOuBuscarCobranca: tool({
      description: 'Cria ou busca uma cobrança no Asaas para o cliente',
      parameters: z.object({
        customer_name: z.string(),
        customer_cpf_cnpj: z.string(),
        customer_phone: z.string(),
        valor: z.number().describe('Valor em reais'),
        descricao: z.string(),
      }),
      execute: async (params) => {
        // tenantId e asaasConfig via closure
        const apiKey = asaasConfig.api_key; // descriptografado em memória
        // 1. Buscar se cliente já existe
        // 2. Criar cliente se não existir
        // 3. Criar cobrança
        // 4. Retornar link de pagamento
      }
    }),
  };
}
```

**2. Registrar no createTenantTools**
```typescript
// tenant/tool-factory.ts
import { createAsaasTools } from '../integrations/asaas';

export function createTenantTools(tenantId: string, config: TenantConfig) {
  const tools = {
    escalarHumano: ...,
    // ... tools core
  };

  if (config.active_tools.includes('google_calendar')) {
    Object.assign(tools, createCalendarTools(tenantId, config.credentials.google_calendar));
  }

  if (config.active_tools.includes('asaas')) {
    Object.assign(tools, createAsaasTools(tenantId, config.credentials.asaas));
  }

  // Nova integração: adicionar bloco if aqui
  // if (config.active_tools.includes('loja_integrada')) { ... }

  return tools;
}
```

**3. Ativar para um cliente (sem código)**
```sql
-- Ativar Asaas para a Julia
UPDATE zenya_tenants
SET active_tools = active_tools || '["asaas"]'::jsonb
WHERE chatwoot_account_id = 'ID_DA_JULIA';

-- Inserir credencial Asaas da Julia
INSERT INTO zenya_tenant_credentials (tenant_id, service, credentials_encrypted)
VALUES ('UUID_JULIA', 'asaas', encrypt('{"api_key": "KEY_DA_JULIA"}'));
```

### CONTRIBUTING.md — estrutura mínima
O arquivo deve cobrir:
1. Pré-requisitos (conta + credenciais)
2. Criar arquivo `integrations/{nome}.ts` com factory function
3. Registrar no `createTenantTools()` com guard `active_tools.includes`
4. Inserir credencial criptografada no banco
5. Ativar no tenant via UPDATE
6. Testar: verificar que tool aparece no contexto LLM apenas para tenant ativo
