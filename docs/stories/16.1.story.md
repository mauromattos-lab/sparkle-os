## Dev Notes

### Drift CRLF vs LF

Aprendizado registrado: comparação md5 direta de `.md` local vs banco Supabase pode dar falso drift porque Git Bash (Windows) normalmente preserva CRLF enquanto banco grava LF. Sempre usar `diff -wB` ou comparar via `gray-matter` parse + `.trim()` antes de concluir que há drift real.

### Rollback operacional (testado via seed idempotente)

Reverter o prompt pra v1 é:
1. `git checkout 8f03b03~1 -- docs/zenya/tenants/fun-personalize/prompt.md` (ou commit anterior à v2)
2. `cd packages/zenya && node --env-file=.env scripts/seed-fun-personalize-tenant.mjs` com env vars corretas
3. Cache de 5min expira automaticamente — próxima mensagem usa v1 de novo

Custo: ~30s. Zero downtime.
