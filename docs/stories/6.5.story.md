# Story 6.5 — Tema Ghost com Identidade Visual Plaka

**Epic:** 6 — Canal de Blog AEO (Ghost CMS)
**Status:** Ready

## Descrição

Como Mauro, quero que o blog Ghost da Plaka tenha a identidade visual da marca — cor de destaque, logo, tipografia e layout refinado — para que visitantes e mecanismos de IA reconheçam que estão num canal oficial da Plaka Acessórios, com aparência profissional e coerente com a loja.

---

## Acceptance Criteria

- [ ] AC1: Cor de destaque (`accent_color`) da Plaka configurada no Ghost via Admin API
- [ ] AC2: Logo da Plaka configurado no Ghost (header do blog exibe a marca)
- [ ] AC3: Título e descrição do site configurados ("Blog Plaka Acessórios" / tagline AEO)
- [ ] AC4: CSS personalizado aplicado via Code Injection — refinamentos de tipografia, espaçamento e cor
- [ ] AC5: Script `packages/brain/scripts/ghost-theme-setup.ts` executa toda a configuração via Admin API (reproduzível)
- [ ] AC6: Blog acessível em `http://187.77.37.88:2368` exibe visual Plaka (validado via screenshot ou browser)

---

## Escopo

**IN:**
- Script `ghost-theme-setup.ts` que configura via Ghost Admin API: accent_color, logo, título, descrição, custom CSS (codeinjection_head)
- CSS personalizado: overrides de CSS variables do tema Casper (cor, font, border-radius)
- Documentação das configurações aplicadas

**OUT:**
- Tema Handlebars customizado (modificação de arquivos `.hbs`) — complexidade alta para ganho marginal
- Configuração de domínio `blog.plakaacessorios.com` — bloqueante externo (DNS)
- Dark mode personalizado
- Imagem de capa/cover do blog — backlog futuro

## Notas Técnicas

### Ghost Admin API — endpoints de configuração

```typescript
// PUT /ghost/api/admin/settings/
// Permite configurar: accent_color, title, description, logo, icon,
//                     codeinjection_head, codeinjection_foot

const settings = [
  { key: 'title', value: 'Blog Plaka Acessórios' },
  { key: 'description', value: '{{tagline de Mauro}}' },
  { key: 'accent_color', value: '{{cor hex de Mauro}}' },
  { key: 'codeinjection_head', value: '<style>/* CSS custom */</style>' },
];
```

### CSS Custom (Casper overrides)

O tema Casper usa CSS variables sobreponíveis via `:root`:

```css
:root {
  --color-accent: {{cor de Mauro}};           /* botões, links, destaque */
  --color-darkgrey: #2d2d2d;                  /* texto principal */
  --font-sans: 'Inter', system-ui, sans-serif; /* tipografia moderna */
}

/* Header com fundo escuro Plaka */
.site-header {
  background: {{cor de Mauro}};
  color: #fff;
}

/* Posts com cards mais limpos */
.post-card {
  border-radius: 12px;
  box-shadow: 0 2px 8px rgba(0,0,0,0.08);
}
```

### Upload de logo via Admin API

```typescript
// POST /ghost/api/admin/images/upload
// Content-Type: multipart/form-data
// Retorna: { images: [{ url: 'https://...' }] }
// Depois: PUT /settings/ com { key: 'logo', value: uploadedUrl }
```
