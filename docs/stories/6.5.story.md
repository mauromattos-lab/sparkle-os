## Descrição

Como Mauro, quero que o blog Ghost da Plaka tenha a identidade visual da marca — cor de destaque, logo, tipografia e layout refinado — para que visitantes e mecanismos de IA reconheçam que estão num canal oficial da Plaka Acessórios, com aparência profissional e coerente com a loja.

- [x] AC1: Cor de destaque (`accent_color`) da Plaka configurada no Ghost via Admin API
- [x] AC2: Logo da Plaka configurado no Ghost (header do blog exibe a marca)
- [x] AC3: Título e descrição do site configurados ("Blog Plaka Acessórios" / tagline AEO)
- [x] AC4: CSS personalizado aplicado via Code Injection — refinamentos de tipografia, espaçamento e cor

- [x] AC6: Blog acessível em `http://187.77.37.88:2368` exibe visual Plaka (validado via HTML — título, logo URL, accent_color e CSS custom confirmados)

## Escopo

**IN:**
- Script `ghost-theme-setup.ts` que configura via Ghost Admin API: accent_color, logo, título, descrição, custom CSS (codeinjection_head)
- CSS personalizado: overrides de CSS variables do tema Casper (cor, font, border-radius)
- Documentação das configurações aplicadas

- Configuração de domínio `blog.plakaacessorios.com` — bloqueante externo (DNS)
- Dark mode personalizado
- Imagem de capa/cover do blog — backlog futuro

## Notas Técnicas

### Ghost Admin API — endpoints de configuração

```typescript
// PUT /ghost/api/admin/settings/
// Permite configurar: accent_color, title, description, logo, icon,
//                     codeinjection_head, codeinjection_foot

const settings = [
  { key: 'title', value: 'Blog Plaka Acessórios' },
  { key: 'description', value: 'Dicas de moda, acessórios e tendências para você se expressar com estilo.' },
  { key: 'accent_color', value: '#0197b2' },
  { key: 'codeinjection_head', value: '<style>/* CSS custom Poppins + Casper overrides */</style>' },
];
```

### CSS Custom aplicado (Casper overrides)

- `@import` Poppins via Google Fonts
- `:root` override de `--gh-font-body`, `--gh-font-heading`, `--color-accent`
- `.gh-head` — fundo teal `#0197b2`
- `.gh-head-logo` — `filter: brightness(0) invert(1)` (logo branco no fundo teal)
- `.post-card` — `border-radius: 12px`, box-shadow, hover lift
- `.post-card-primary-tag` — cor rosa `#d94f73`
- `.gh-btn` — botões teal com `border-radius: 8px`

### Upload de logo via Admin API

```typescript
// POST /ghost/api/admin/images/upload
// Content-Type: multipart/form-data (logo baixado do CDN NuvemShop)
// Retorna: { images: [{ url: 'https://...' }] }
// Depois: PUT /settings/ com { key: 'logo', value: uploadedUrl }
```
