# Story 5.5 — Integração Google Drive

**Epic:** 5 — Content Engine: Automação e Integração
**Status:** Ready for Review
**Prioridade:** P2
**Executor:** @dev
**Estimativa:** S
**Depende de:** 5.1

---

## Descrição

Como Mauro, quero que Vista consiga acessar automaticamente a pasta do Drive da Luiza para selecionar a imagem do Pinterest, sem que eu precise fazer download e upload manual de imagens.

---

## Acceptance Criteria

- [ ] AC1: Vista acessa a pasta compartilhada do Drive via Google Drive API
- [ ] AC2: Vista lista as imagens disponíveis e seleciona a mais relevante ao tópico do post
- [ ] AC3: Imagem selecionada é disponibilizada para a Story 5.4 (Pinterest) sem download manual
- [ ] AC4: Se pasta estiver vazia ou inacessível, Vista usa placeholder e registra aviso no Cockpit
- [ ] AC5: Credenciais Google Drive armazenadas como variáveis de ambiente (`GOOGLE_DRIVE_FOLDER_ID`, `GOOGLE_SERVICE_ACCOUNT_KEY`)

---

## Escopo

**IN:**
- Leitura da pasta compartilhada via Google Drive API
- Seleção de imagem por relevância (nome do arquivo ou ordem cronológica como fallback)
- Disponibilização da imagem para a pipeline de publicação

**OUT:**
- Upload de novas imagens para o Drive
- Organização automática de pastas
- Análise de conteúdo da imagem por IA (fase futura)

---

## Notas Técnicas

- Google Drive API v3: `files.list` com `folderId`
- Autenticação via Service Account (mais simples que OAuth para automação)
- Luiza compartilha a pasta com o e-mail da Service Account
- Seleção: imagem mais recente como default, refinamento por nome como evolução
