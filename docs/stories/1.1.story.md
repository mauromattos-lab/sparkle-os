## Estrutura Alvo (definida por @architect)

```
sparkle-os/
├── packages/
│   ├── core/          # SparkleOS Core — Context Store, ADR Registry, Internal API (Hono)
│   ├── brain/         # Collective Brain — ingestão, validação, aplicação
│   └── shared/        # Tipos TypeScript compartilhados entre packages
├── organs/
│   └── zenya/         # Zenya Adapter — n8n + Chatwoot + Z-API (Epic 2)
├── apps/
│   └── piloting/      # Piloting Interface — Next.js 15 cockpit (Epic 4)
├── infra/             # Coolify configs, Docker Compose, GitHub Actions
├── docs/
│   ├── stories/       # Stories AIOS (este diretório)
│   ├── adrs/          # ADR Registry (Story 1.4)
│   ├── sops/          # SOPs (Story 1.7)
│   ├── prd.md
│   ├── architecture.md
│   └── brief.md
├── .aios/             # Runtime AIOS (handoffs, logs, estado de sessão)
├── pnpm-workspace.yaml
├── package.json       # Root package.json com scripts globais
├── tsconfig.base.json # TypeScript config base compartilhada
├── .gitignore
└── README.md
```
