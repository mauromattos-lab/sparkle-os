## Descrição

Como Mauro, quero que os posts publicados no Ghost incluam estrutura AEO expandida — FAQ block com JSON-LD FAQPage e Schema BlogPosting completo — para que os artigos da Plaka sejam indexados corretamente pelos motores de IA (Perplexity, ChatGPT Search, Google AIO) e apareçam em respostas diretas com rich snippets.

- [x] AC2: `BlogPosting` expandido inclui: `headline`, `author`, `publisher`, `datePublished`, `inLanguage`, `url` (blogUrl do post), `wordCount` (estimado a partir do HTML), `articleSection` (derivado do topic)

## Notas Técnicas

### Funções a adicionar em `ghost-publisher.ts`

```typescript
// Extrai até N pares FAQ de H2/H3 + parágrafo seguinte do HTML
export function extractFaqItems(html: string, maxItems = 5): Array<{ question: string; answer: string }>

// Estima wordCount removendo tags HTML e contando palavras
export function estimateWordCount(html: string): number

// buildJsonLd expandido — mantém retrocompatibilidade
export function buildJsonLd(
  title: string,
  datePublished: string,
  options?: {
    url?: string;
    wordCount?: number;
    articleSection?: string;
    faqItems?: Array<{ question: string; answer: string }>;
  }
): string
```

### Schema JSON-LD resultante

```json
[
  {
    "@context": "https://schema.org",
    "@type": "BlogPosting",
    "headline": "{{título}}",
    "author": { "@type": "Organization", "name": "Plaka Acessórios" },
    "publisher": { "@type": "Organization", "name": "Plaka Acessórios" },
    "datePublished": "{{ISO date}}",
    "inLanguage": "pt-BR",
    "url": "{{blogUrl}}",
    "wordCount": 850,
    "articleSection": "{{topic}}"
  },
  {
    "@context": "https://schema.org",
    "@type": "FAQPage",
    "mainEntity": [
      {
        "@type": "Question",
        "name": "{{pergunta extraída do H2/H3}}",
        "acceptedAnswer": {
          "@type": "Answer",
          "text": "{{parágrafo seguinte ao heading}}"
        }
      }
    ]
  }
]
```

### Lógica de extração FAQ

```
1. Parse HTML via regex: encontra pares (H2|H3) + <p> imediatamente seguinte
2. Strip tags HTML das strings encontradas
3. Filtra pares onde question.length > 10 AND answer.length > 20
4. Retorna até maxItems pares
5. Se len(pares) < 3 → retorna [] → FAQPage omitido do schema
```

### Atualização da chamada em `publishToGhost`

```typescript
const htmlBody = bodyHtml; // já gerado
const faqItems = extractFaqItems(htmlBody);
const wordCount = estimateWordCount(htmlBody);

codeinjection_head: buildJsonLd(title, now, {
  url: undefined,   // preenchido após POST (blogUrl ainda não disponível)
  wordCount,
  articleSection: post.topic ?? undefined,
  faqItems,
}),
```

> **Nota:** `url` no BlogPosting não pode ser preenchido antes da criação (Ghost retorna a URL no response). Deixar `undefined` neste campo é aceitável para AEO — `datePublished` e `headline` são os campos críticos.
